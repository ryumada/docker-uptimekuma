# The data of Uptime Kuma is stored in this directory.
- This directory is mounted to the container at /app/data.
- If you want to backup the data, you can backup this directory.
- If you want to move the data to another directory, you can move this directory.
- If you want to delete the data, you can delete this directory.
- If you want to restore the data, you can restore this directory.
